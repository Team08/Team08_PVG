1.0 Installationsbeskrivning

Progam som ingår:
Registerprogram, Sorteringsprogram

Hur körs dom:
Registerprogrammet är grafiskt, medan Sorteringsprogrammet körs från terminalen.
Båda programmen är jar-filer. (För att starta en .jar fil behöver Java Runtime Environment vara installerad)
För att starta Registreringsprogrammet räcker det med att dubbelklicka på programmets ikon, medan du måste gå in i terminalen, för att kunna öppna sorteringsprogrammet.
Parametrar:
Sorteringsprogrammet behöver fyra parametrar - namnet på starttidsfilen, måltidsfilen, namnfilen samt resultatsfilen.
